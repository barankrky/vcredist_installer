# Visual C++ Redistributables All-in-One Installer for Windows

- Completely silent installation.
- Automatically install for 32-bit or 64-bit computers.
- Tested on Windows 10

## Silent Version
- Installation becomes completely silent and progress will show up in the notification area.
  
![image](https://github.com/barankrky/vcredist_installer/assets/53481486/136c62c6-b2d9-43be-9325-b779c6ff844b)

## Console Version
- Installing progress will show up in a command prompt.
