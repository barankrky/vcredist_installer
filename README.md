# Visual C++ Redistributables All-in-One Installer for Windows

- Completely silent installation.
- Automatically install for 32-bit or 64-bit computer.
- Tested on Windows 10

## How to install
- Just download latest release and extract to a folder. Run vcredist_installer.exe and you're good to go.
